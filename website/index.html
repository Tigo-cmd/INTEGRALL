Why Integrall

Write logic, not boilerplate

Integrall abstracts away hardware complexity so you can focus on building what matters.

⚡

One-Line Hardware

Control relays, LCDs, servos, and sensors with single function calls. No more 50-line initialization routines — just integrall.enableRelay(5) and you're done.

🔄

Non-Blocking by Default

Every operation uses millis() timing. Servo sweeps, buzzer patterns, LCD scrolling — all run in the background while your WiFi and sensors stay responsive.

☁️

Professional Cloud Sync

Works offline or online. Launch in AP Mode for local control, or Station Mode with a one-line telemetry sync: integrall.sendTelemetry("temp", 25).

Hardware Modules

10 modules. One include.

Enable only what you need. Each module compiles to zero overhead when disabled — perfect for memory-constrained boards.

🔌

Relay Control

Up to 4 relays with safety timeouts, interlock groups, and physical button triggers.

int r = integrall.enableRelay(5, true, "Pump");
📺

LCD Display

I2C 16x2 LCD with auto-scrolling for long text, custom characters, and cursor control.

integrall.lcdPrint("Hello World!", 0, 0);
🖥️

OLED Display

SSD1306 128x64 OLED with text, labeled values, and progress bar drawing.

integrall.oledPrintValue("Temp: ", 24.5, 0);

Servo Motors

Up to 4 servos with instant set, non-blocking sweep, smooth easing, and analog mapping.

integrall.easeServo(arm, 180, 15);

Sensors

Ultrasonic distance, PIR motion, analog %, LDR light, and DHT temperature/humidity.

Matrix Keypad

4x4 and 4x3 keypads with buffer capture, backspace, submit, and PIN validation.

Buzzer Alerts

Non-blocking beeps, patterns, and melodic feedback — alert, success, and failure tones.

RGB LED

Common anode/cathode support with named color presets and non-blocking blink.

Get Started

Up and running in 2 minutes

Install the library, include the header, and start building.

; platformio.ini
[env:esp32dev]
platform = espressif32
board = esp32dev
framework = arduino
lib_deps =
    bblanchon/ArduinoJson @ ^6.21.3
    ; Copy the Integrall folder into your project's lib/ directory
    ; or install from the release .zip

Place the Integrall/ folder inside your project's lib/ directory, then build.

// 1. Open Arduino IDE
// 2. Sketch → Include Library → Add .ZIP Library...
// 3. Select the Integrall_v1_0.zip file
// 4. Install "ArduinoJson" from Library Manager
// 5. Done! Now use it:

#define INTEGRALL_ENABLE_RELAY
#include <Integrall.h>

Integrall::System integrall;

void setup() { integrall.begin(); }
void loop()  { integrall.handle(); }

Install additional libraries (LCD, Keypad, Servo, etc.) from the Library Manager based on which modules you enable.

# Clone the repository
git clone https://github.com/emmanueltigo/integrall.git

# Copy the library into your Arduino libraries folder
cp -r integrall/firmware/Integrall ~/Arduino/libraries/

# Or for PlatformIO, copy into your project
cp -r integrall/firmware/Integrall your-project/lib/

See DEPENDENCIES.md for the full list of optional libraries per module.
